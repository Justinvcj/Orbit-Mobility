## 📖 Overview
**Orbit Mobility** is the sleek, performant mobile frontend for the Equinox Ride-Hailing platform. Built with Flutter, it ensures a native-like, unified experience across both iOS and Android devices. 

Designed for high-frequency updates, Orbit Mobility communicates with the backend via WebSockets to provide real-time vehicle tracking, instant ride matching, and a fluid, responsive user interface.

## ✨ Key Features
- **🌐 Cross-Platform Performance:** A single Flutter codebase delivering native performance on both iOS and Android.
- **📍 Live Geospatial Tracking:** Renders continuous, smooth map interpolation of driver locations using real-time WebSocket streams.
- **💳 Digital Wallet Integration:** Securely displays and manages user wallet balances and transaction histories.
- **🚀 Instant Dispatch UI:** A highly responsive ride request and acceptance interface with minimal latency.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0+)
- Dart SDK
- Android Studio / Xcode for emulation

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Justinvcj/Orbit-Mobility.git
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
